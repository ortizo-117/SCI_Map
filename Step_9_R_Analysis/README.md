**R Analysis**

### Step 9: Statistical Analysis for Structural Data

Run the statistical analysis on two complementary data tracks:

1. **BrainCharts normative z-scores** from `Step_3_BrainCharts_Normative_Modeling/run_braincharts_normative_models.py`.
2. **Raw FreeSurfer morphometry** from `Step_8_Data_Aggregation`.

This lets the SCI_MAP meta-analysis compare regional normative deviations against raw cortical and subcortical morphometric differences.

### Required software

Use a working R/RStudio installation with:

- `tidyverse`
- `data.table`
- `multcomp`
- `effsize`
- `writexl`
- `ggpubr`
- `esvis`

### Expected inputs

| File | Source | Contents |
|:--|:--|:--|
| `braincharts_zscores.csv` | Step 3 | Metadata plus regional BrainCharts z-score columns |
| `dk_all_stats.csv` | Step 8 | Raw DK cortical thickness, surface area, gray volume, and aseg subcortical volumes |
| `icv_data.csv` | Step 8 | Intracranial volume values |
| `cortical_thickness.csv` | Step 8 | Mean left/right cortical thickness |
| `wm_volumes.csv` | Step 8 | Regional white matter volumes |
| `metadata.csv` | Site/team | Subject-level age, sex, cohort, pain status, injury variables, and scanner/site labels |

The metadata subject IDs must match the IDs in the BrainCharts and FreeSurfer output tables.

### Analysis plan

Run the same cohort-comparison framework on:

- BrainCharts z-score regions.
- Raw cortical thickness regions.
- Raw cortical surface area regions.
- Raw cortical gray matter volume regions.
- Raw subcortical volume regions.
- Raw white matter volume regions.
- Summary measures such as ICV and mean hemispheric cortical thickness when relevant.

For each track, estimate group differences and effect sizes for:

- SCI with neuropathic pain versus healthy controls.
- SCI without neuropathic pain versus healthy controls.
- SCI with neuropathic pain versus SCI without neuropathic pain.
- All SCI participants versus healthy controls.

If a site does not have all cohorts, run only the comparisons supported by that site's data and document which comparisons were omitted.
